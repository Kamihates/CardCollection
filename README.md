# CardCollection

### Présentation du projet
***

Le projet consiste à créer un site web html/css de collection de jeu de carte sur un thème imposés (parmi une liste)

### Installation
***

```
- Installer XAMPP
- Run XAMPP (apache)
- Ouvrir une page internet et rentrer localhost
```
